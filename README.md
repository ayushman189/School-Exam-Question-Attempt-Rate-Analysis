# School Exam Question Attempt Rate Analysis

## Project Overview
This project analyzes question-wise attempt rates of students in a school examination.
The goal is to identify the difficulty level of questions based on how many students attempted them.

This is a Statistics Minor Semester Project implemented using Python and Pandas in Jupyter Notebook.
